ArrayList和LinkedList是如何实现的？

分类：JDK， 作者： 徐梦旗， 发布于： 2023-09-15 20:00， 字数：396，预计阅读：1分钟
#源码

1. ArrayList详解

1.1. 为什么要有ArrayList？

如果想要维护多个元素，可以使用数组，那为什么还要有ArrayList？相比于数组，ArrayList有以下几个区别：

  1. 使用数组时必须指定数组大小，ArrayList会自动扩容，不需要指定大小；
  2. ArrayList是面向对象封装的，提供封装的操作方法。

1.2. ArrayList底层是如何实现的？

  1. 从继承关系上看，ArrayList实现了List接口和RandomAccess接口；
  2. 从底层数据结构上看，ArrayList使用了动态数组；
  3. 从线程安全上看，ArrayList是线程不安全的，采用fast-fail机制。

1.3. ArrayList是如何扩容的？

默认容量为10.在新增元素时，会判断容量是否足够，如果不够，数组会扩容为原数组大小的1.5倍。

1.4. ArrayList常用的API有哪些？

add O(1)或O(n)
remove O(n)
set O(1)
get O(1)

2. LinkedList详解

2.1. LinkedList是如何实现的？

底层使用链表

2.2. LinkedList常用的API有哪些？

add O(1) 或 O(n)
remove O(1) 或 O(n)
set O(n)
get O(n)

2.3. LinkedList和ArrayList有什么区别？

基于链表实现，基于数组实现，无需动态扩容；
对于操作方法，ArrayList是随机访问，也就是O(1)

2.4. Vector底层是如何实现的？

基于数组，会动态扩容，使用内置锁是线程安全的。

2.5. Vector和ArrayList有什么区别？

线程安全上

2.6. Stack底层是如何实现的？

基于Vector实现。
